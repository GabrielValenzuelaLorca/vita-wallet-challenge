    2. Install dependencies:
       - Production: `antd`, `@ant-design/icons`, `@tanstack/react-query`, `zod`, `react-router-dom`
       - Dev: `@typescript-eslint/eslint-plugin`, `@typescript-eslint/parser`, `prettier`

    3. Configure tsconfig.json with strict mode:
       ```json
       {

           "strict": true,
           "noImplicitAny": true,

           "noUnusedLocals": true,
           "noUnusedParameters": true,
           "noFallthroughCasesInSwitch": true,
           "forceConsistentCasingInFileNames": true,
           "paths": { "@/*": ["./src/*"] }
         }
       }
       ```
       Ensure tsconfig.app.json inherits from tsconfig.json and includes src/.

    4. Configure vite.config.ts:
       - Add React plugin
       - Add path alias: `@/` -> `./src/`
       - Set server port 5173
       - Set proxy for `/api` to `http://localhost:3000` (Rails backend) to avoid CORS in dev

    5. Configure eslint.config.js (flat config format):
       - Extend typescript-eslint recommended
       - Set `@typescript-eslint/no-explicit-any` as "error" (NOT warn, ERROR)
       - Set `@typescript-eslint/no-unused-vars` as "error" with argsIgnorePattern: "^_"
       - Add React and React Hooks plugins

    6. Configure main.tsx:
       - Import React StrictMode
       - Import QueryClient and QueryClientProvider from @tanstack/react-query
       - Create QueryClient with default options:
         - queries: { retry: 1, staleTime: 30000 }
         - mutations: { retry: 0 }
       - Import BrowserRouter from react-router-dom
       - Wrap App in: StrictMode > QueryClientProvider > BrowserRouter
       - Do NOT import any CSS files (antd v5+ uses CSS-in-JS)

    7. Configure App.tsx:
       - Import AuthProvider from contexts/AuthContext
       - Import ConfigProvider from antd (light theme default, no custom theme needed)
       - Wrap routes in: ConfigProvider > AuthProvider
       - Add placeholder routes using react-router-dom Routes/Route:
         - `/login` -> LoginPage (placeholder)
         - `/` -> DashboardPage (placeholder, wrapped in ProtectedRoute)
         - `*` -> Navigate to `/`
       - These are shells that will be fleshed out in Phase 2

    8. Remove all default Vite template CSS files (App.css, index.css). antd handles all styling.

       export const apiMetaSchema = z.object({

         per_page: z.number().optional(),
         total: z.number().optional(),
       });

       export function apiEnvelopeSchema<T extends z.ZodTypeAny>(dataSchema: T) {
         return z.object({
           data: dataSchema,
           meta: apiMetaSchema.optional(),
         });
